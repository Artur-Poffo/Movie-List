# Movie-List:

> Status: Versão 1.0.0

Lista de filmes que consulta dados de uma API externa do TMDB, te permite ver informações dos filmes mais populares do momento e pesquisar por filmes.

## Tecnologias:
- React.js
- Next.js (com SSG e ISR)
- styled-components
- ESlint
- Node.js
- JavaScript
